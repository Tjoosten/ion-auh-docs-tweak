Function: remove_from_group()

Remove user form group(s)

Parameters:
  • Group_id - NULL, integer of array. Null will remove the user from all groups. REQUIRED
  • User_id - integer. REQUIRED
Return:
  • boolean. TRUE if the user was removed from group(s). FALSE if the user is not removed from group(s).

Usage:

{% highlight php %} // pass an array of group ID's and user ID $this->ion_auth->remove_from_group(array('1', '3', '6'), $user_id); // pass a single ID and user ID $this->ion_auth->remove_from_group(1, $user_id); // pass NULL to remove user from all groups $this->ion_auth->remove_from_group(NULL, $user_id); {% endhighlight %}
