Function: update_group()

Update details of a group.

Parameters:
  • group_id - int REQUIRED
  • group_name - string REQUIRED
  • group_description - string OPTIONAL
Return:
  • boolean. TRUE if the group was updated, FALSE if the update failed.

Usage:
