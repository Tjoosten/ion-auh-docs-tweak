Function: increase_login_attempts()

If login attempt tracking is enabled, records another failed login attempt for this this identity or IP address. This method is automatically called during the login method if the login failed.

Parameters:
  • Identity - string. REQUIRED

Usage:

{% highlight php %} $identity = 'ben.edmunds@gmail.com'; $password = '12345678'; if ($this->ion_auth->login($identity, $password) == FALSE) { $this->ion_auth->increase_login_attempts($identity) } {% endhighlight %}
