Function: update()

Update a user.

Parameters:
  • Id - integer. REQUIRED
  • Data - multidimensional array. REQUIRED
Return:
  • boolean - TRUE is the user was successfully updated. FALSE is the user was not updated.

Usage:

{% highlight php %} $id = 12; $data = array( 'first_name' => 'Ben', 'last_name' => 'Edmunds', 'password' => '123456789', ); $this->ion_auth->update($id, $data) {% endhighlight%}
