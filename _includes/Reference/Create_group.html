Function: create_group()

Create a group.

Parameters:
  • group_name - string. REQUIRED
  • group_description - string OPTIONAL
Return:
  • brand new group_id if the group was created, FALSE if the group creation failed.

Usage:

{% highlight php %} // pass the right arguments and it's done $group = $this->ion_auth->create_group('new_test_group', 'This is a test description'); if(!$group) { $view_errors = $this->ion_auth->messages(); } else { $new_group_id = $group; // do more cool stuff } {% endhighlight %}
