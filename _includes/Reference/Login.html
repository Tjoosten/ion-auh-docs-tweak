Function: login()

Logs the user into the system.

Parameters:
  • Username - string. Usally username of email but depends on your config. REQUIRED
  • Password - string. REQUIRED
  • Remember - boolean. TRUE sets the user to be remembered if enabled in the config. OPTIONAL
Return:
  • boolean - TRUE if the user was successfully logged in. FALSE if the user was not logged in.

Usage:

{% highlight php %} $identity = 'ben.edmunds@gmail.com'; $password = '12345678'; $remember = TRUE; // remember the user $this->ion_auth->login($identity, $password, $remember); {% endhighlight %}
