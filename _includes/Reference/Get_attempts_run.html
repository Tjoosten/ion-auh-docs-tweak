Function: get_attempts_run()

Returns the number of failed login attempts for this identity or IP address.

Parameters:
  • Identity - string. REQUIRED
Return:
  • int. The number of failed login attempts for this identity or IP address.

Usage:

{% highlight php %} $identity = 'ben.edmunds@gmail.com'; $num_attempts = $this->ion_auth->get_attempts_num($identity); {% endhighlight %}
