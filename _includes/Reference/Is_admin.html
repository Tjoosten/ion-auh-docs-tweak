Function: is_admin()

Check to see if the currently logged in user is an admin.

Parameters:
  • id - integer. if a user id of the currently logged in user will be used. OPTIONAL
Return
  • boolean. TRUE if user is an admin. FALSE if the user is not an admin.

Usage:

{% highlight php %} if (!$this->ion_auth->is_admin()) { $this->session->set_flashdata('message', 'You must be an admin to view this page'); redirect('welcome/index'); } {% endhighlight %}
