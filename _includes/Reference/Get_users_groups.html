Function: get_users_groups()

Get all groups a user is part of.

Parameters:
  • Id - integer. If u user id is not passed the id of the currently logged in user will be used. OPTIONAL

Return:

{% highlight php %} stdClass Object ( [id] => 1 [name] => admins [description] => Administrator ) {% endhighlight%}

Usage:

{% highlight php %} $user_groups = $this->ion_auth->get_users_groups($user->id)->result(); {% endhighlight %}
