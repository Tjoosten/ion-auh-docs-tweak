Function: delete_group()

Remove a group. Removes the group details form the configured groups table. Users belonging to the group are stripped. of this status (references to this group are removed form users_group)), but user data itself remains untouched.

Parameters:
  • group_id - int REQUIRED
Return:
  • boolean. TRUE is the group was deleted, FALSE if the delete failed.
