Function: add_to_group()

Add user to group

Parameters:
  • Group_id - integer or array. REQUIRED
  • User_id - integer REQUIRED
Return:
  • boolean. TRUE if the uer was added to group(s). FALSE if the user is not added to group(s).

Usage:

{% highlight php %} // pass an array of group ID's and user ID $this->ion_auth->add_to_group(array('1', '3', '6'), $user_id); // pass a single ID and user ID $this->ion_auth->add_to_group(1, $user_id); {% endhighlight %}
