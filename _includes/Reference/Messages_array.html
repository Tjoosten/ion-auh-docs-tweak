Function: messages_array()

Get messages as an aray.

Parameters:
  • Langify - boolean. TRUE means that the messages will be langified. OPTIONAL
Return:
  • array

Usage:

{% highlight php %} $id = 12; $data = array( 'first_name' => 'Ben', 'last_name' => 'Edmunds', ); if ($this->ion_auth->update_user($id, $data)) { $messages = $this->ion_auth->messages_array(); foreach ($messages as $message) { echo $message; } } else { $errors = $this->ion_auth->errors_array(); foreach ($errors as $error) { echo $error; } } {% endhighlight %}
